# HTML-CSS

Welcome to the central tracking directory for HTML and CSS created by **Sanaa Alalyan**. This workspace houses design modules, styling experiments, and structure blueprints focused on foundational web layouts.

## Projects Documented

### 1. Portfolio Site (This Workspace)
A highly distinct, vibrant, and interactive single-page portfolio interface built using strict native markup and stylistic parameters. 

* **Integrated Fixed Navigation**: A sticky header bar showcasing exact hover parameters, custom block elements, and interactive responsive offsets.
* **Two-Column Structural Layout**: Balanced content arrays using clean structural column blocks that feature controlled pixel height limits, structural width configurations, and designated font accents.
* **Multimedia Integration**: Dynamic looped video component running completely muted inside a text overlay hero section, built alongside layered graphic cards showcasing CSS layout properties, rounded geometric corner styles, and graphic hue filters.
* **Data Layouts**: A custom analytics framework structured via tabular elements, incorporating uniform row padding borders and unique header accents.

### 2. Bootstrap Project Expansion (Forthcoming Deployment)
The secondary stage of this repository focuses on migration into responsive grid utilities. Using the Bootstrap framework, this component transitions manual CSS sizing calculations into modern, auto-scaling grids, card structures, and utility classes.
